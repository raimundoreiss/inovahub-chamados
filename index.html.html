Firebase não configurado — siga o guia abaixo

1. Acesse console.firebase.google.com e crie um projeto.
2. Vá em Firestore Database → Criar banco → Modo de teste.
3. Vá em Configurações do projeto → Seus apps → Web (</>) → Registrar app.
4. Copie o objeto firebaseConfig e cole no início deste arquivo HTML, substituindo os valores COLE_AQUI.
5. Salve o arquivo e hospede no GitHub Pages (veja o guia de hospedagem incluso).
